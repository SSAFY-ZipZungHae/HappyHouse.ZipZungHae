# ZipZungHae

삼성청년SW아카데미 - 1학기 HappyHouse 관통 프로젝트(2021 상반기)

## 개발 도구

|도구|버전|설명|
|:---:|:---:|:---:|
|STS|3.9.14|Backend+Frontend|
|VSCode|1.56.2|Frontend|

## 개발 환경

|환경|버전|설명|

>   * 회원 목록
>

>   
> * 아파트 검색

>   * 아파트 상세 정보
>
> * 추가 정보 제공
>   * 상권 정보 제공
>   * 코로나 진료 선별소 정보 제공
